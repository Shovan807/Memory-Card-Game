# Memory-Card-Game

The Memory Card Game is a simple and interactive game that tests the playerâ€™s memory and concentration skills. The objective of the game is to find and match pairs of identical cards by flipping them over two at a time.

The game starts with all cards placed face down. The player selects two cards per turn:

If the cards match, they remain visible.

If not, they flip back after a short delay.

The game continues until all card pairs are matched. Fewer moves and faster completion indicate better performance.

Features

Simple and user-friendly interface

Card flip animations

Match detection logic

Restart/reset option

This project is ideal for beginners to understand basic game logic, DOM manipulation, and event handling in JavaScript.
